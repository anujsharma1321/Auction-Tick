## 🚀 Getting Started

<!-- Prerequisites -->
### 🔧 Prerequisites

- [Composer](https://getcomposer.org/)
- [MySQL](https://www.mysql.com/)

<!-- Run Locally -->
### 🏃 Run Locally

Clone the project

```bash
  git clone https://github.com/anujsharma1321/Auction-Tick/
```

Go to the project directory

```bash
  cd auctionx https://github.com/anujsharma1321/Auction-Tick/main.php
```

Define your config

```bash
  cp src/config.example.php src/config.php
```

Import the database schema

```bash
  mysql -u your_username -p < path/to/auctionx.sql
```

Generate the autoload file

```bash
  composer dump-autoload
```

Start the php server

```bash
  php -S localhost:5000 -t public
```

Open http://localhost:5000/public in your web browser
